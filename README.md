# cuda-x-crunch

Run

```
cmake .
```

to start development (generate Makefile or Visual Studio solution).

On Windows use:

```
build_release_win.bat
```

You may have to fix msbuild path in the script if needed.

## Performance results


RTX 3060: 350 Mh/s

With parameters: -k 256 -g 10000 -r 2000
RTX 3090: 990 Mh/s
RTX 4090: 2320 Mh/s
